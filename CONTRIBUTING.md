# Contributing To Supered

Thanks for helping improve Supered.

## Local Checks

Run the full check set before opening a pull request:

```bash
npm test
npm run validate
npm run smoke-install
npm run verify-package
npm run verify-site
```

## Skill Changes

- Keep skill content original.
- Keep skill files concise and action-oriented.
- Update tests and plugin metadata when adding or removing skills.
- Update docs when installation or host compatibility changes.

## Pull Requests

Include:

- what changed
- why it changed
- how you verified it
- screenshots when the landing page changes
