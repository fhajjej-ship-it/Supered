Install

Install in Codex.

Add the public GitHub marketplace source, then install Supered from that source. Use npx when you only want the standalone skill installer.

codex plugin marketplace add fhajjej-ship-it/Supered --ref v0.6.2
codex plugin add supered@supered
codex plugin list | grep 'supered@supered'

npx supered@latest install --target codex
npx supered@latest doctor --target codex
