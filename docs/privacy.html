Policy

Privacy Policy

Supered is a public, open-source workflow plugin and skill bundle for coding agents.

Data Collection

Supered does not run a hosted service, does not include an app connector, and does not collect personal data through the plugin bundle.

Local Use

The Supered CLI copies skill files into local agent-host folders and can inspect or repair those local Supered-managed files. It does not send local project contents to Supered.

External Services

Using Codex, npm, GitHub, or another agent host may involve those services processing data under their own policies. Supered does not control those services.

Contact

Questions and security reports can be opened through the public GitHub repository.
